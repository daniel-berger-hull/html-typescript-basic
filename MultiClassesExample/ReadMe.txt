This projects show a simple multiple level of import/exports in TypeScript.
It creates an object (ItemGrid.ts) that is self contain an item container (DataBank.ts) and a button, 
on which the user clicks, and the content of the ItemGrid will be displayed (an array of strings)

compile with the command (from the root of the project)

tsc

Note: the tsconfig.json file has the config to locate the Typescript files (./src)
